Proyecto Cisco Packet tracer

Explicacion

El proyecto desarrollado en Cisco Packet Tracer consiste en el diseño y simulación de una red empresarial para la organización 4Event C.A., donde se planificó tanto la distribución física como lógica de la infraestructura de red, tomando como base la sede principal ubicada en Chacao y su Piso 1, en el cual se identificaron áreas como aulas, recepción, dirección y cuarto de servidores para una correcta ubicación de los equipos; la red fue diseñada bajo una topología jerárquica con capas de núcleo, distribución y acceso, permitiendo una interconexión eficiente entre las diferentes sedes y academias mediante routers, switches y enlaces WAN, además de la implementación de redes cableadas e inalámbricas y la segmentación por VLANs para mejorar la seguridad y el rendimiento; finalmente, el uso de Cisco Packet Tracer permitió validar la conectividad, el correcto funcionamiento de los dispositivos y la escalabilidad de la red propuesta.

Proyecto Servidor Linux casero

Explicacion

El proyecto de **Servidor Linux casero** consistió en la implementación y configuración de un servidor funcional utilizando un sistema operativo Linux, enfocado en brindar servicios básicos de red y administración, donde se configuraron componentes como el servidor web (Apache y Nginx), servicios de red como DHCP, y archivos de configuración esenciales, además del monitoreo del rendimiento y tráfico del sistema mediante herramientas gráficas de estadísticas; el proyecto permitió comprender el funcionamiento interno de un servidor, la gestión de procesos, logs y servicios, así como la aplicación de buenas prácticas de administración, demostrando que es posible montar y administrar un servidor eficiente utilizando recursos locales y software libre.
